    바리스타 및 방과후 강사 경험으로 체득한 책임감과 유연한 소통 능력을 바탕으로 팀의 신뢰를 쌓아가며, 귀사의 서비스 운영에 기여하는 개발자로 성장하겠습니다.<br/>
    Java, Spring, REST API, DBMS 등 주요 스택을 교육 과정과 독학으로 학습하고, 웹 서비스를 설계·구현해 본 경험이 있습니다. <br/>
    특히 크라우드 펀딩 플랫폼 팀 프로젝트에서는 Spring Boot 기반 REST API를 활용하여 핵심 기능인 '프로젝트 생성-심사' 도메인의 CRUD와 검증 로직 구현을 담당했습니다.<br/>
    이 과정에서 로그 분석과 관찰 결과를 기반으로 복잡한 상태 전이 오류를 해결하며, 실무에서 요구되는 안정적인 시스템 구축 역량을 길렀습니다.
